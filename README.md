# IPL 2023 Data Analysis Dashboard

This project analyzes IPL 2023 match data using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Visualization

## Analysis Performed
- Home vs Away Wins
- Toss Impact on Match Results
- Batting First vs Bowling First Performance
- Team Win Distribution
- Win Percentage by Team

## Key Insights
- Away teams won more matches than home teams.
- Toss did not strongly impact match results.
- Teams batting first had a slight advantage.
- Gujarat Titans recorded the highest win percentage.
